Title: FreshDirect DBMS: Scalable E-Commerce Database ArchitectureOverview

This project is a fully normalized Relational Database Management System (RDBMS) designed for "FreshDirect," an expanding UK-based grocery delivery service. The objective was to transition the company from ad-hoc data storage to a centralized, structured database to ensure data integrity, streamline logistics, and unlock strategic business intelligence.Tech StackDatabase: MySQL Design: Entity-Relationship Modeling, 3rd Normal Form (3NF) Security: Designed for GDPR compliance, supporting data minimization and right to erasure Database Architecture

The database is structured in the Third Normal Form (3NF) to eliminate data redundancy and prevent update anomalies. Designed with scalability in mind, the current schema paves the way for future automated supply chain triggers and API integrations for mobile applications.
